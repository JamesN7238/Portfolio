This Website

I built this website portfolio using HTML, CSS, and JavaScript. It’s hosted on GitHub Pages, allowing me to fully control and showcase my projects with smooth transitions, a responsive design, and custom animations.

Jarvis

Jarvis is my personal AI assistant inspired by Tony Stark’s JARVIS. It includes voice control, live scheduling, and a futuristic UI. The interface is fully editable and controlled by the AI in real time.

AI Chatbot

I created a chatbot using Ollama and LLama3. It runs entirely on my own machine and can have full conversations and provide contextual responses using advanced local inference. I also included a light/dark theme option to make it easier on the eyes.

Arduino Motion Tracker

I connected an accelerometer to an Arduino to create a motion tracking system. The movement is streamed to my laptop in real time and visualized using Python and Processing for a seamless hardware-to-software bridge.
