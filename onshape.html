OnShape Projects

Toy Train Project

This project is a CAD design of a toy train and was built using OnShape. It demonstrates use of the core features such as extrusions, revolves, and assemblies. The toy train design taught me constraint-based modeling and how to manage part relationships within OnShape assemblies. It also helped me learn how to design different fasteners on my own.

Motorcycle Clutch Project

This project showcases a detailed 3D CAD model of a multi-plate motorcycle clutch system which was designed using OnShape for both functional accuracy and visual clarity. The assembly includes key components such as the clutch basket, inner hub, friction and steel plates, pressure plate, springs, and fasteners. The exploded view illustrates the internal mechanics, assembly sequence and mechanical interaction.
